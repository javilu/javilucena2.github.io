Ejercicio 1

Busque la lista de atributos admitidos para la etiqueta area.

  • alt: Especifica texto alternativo para el area.
  • coords: Especifica coordenadas para el area.
  • download: Especifica el objetivo que se va a descargar cuando el usuario pinche en el hipervinculo.
  • href: Especifica el objetivo del hipervinculo.
  • hfreflang: Especifica el lenguaje de la URL.
  • media: Especifica el dispositivo/media para el cual está optimizada la URL.
  • rel: Especifica la relación entre el documento actual y la URL de destino.
  • shape: Especifica la forma del area.
  • target: Especifica donde se abrira la URL.
  • type: Especifica el tipo de media de la URL.

details: Especifica detalles adicionales que el usuario puede ver u ocultar bajo demanda.

summary: Representa un sumario, frase o leyenda sobre el resto del contenido.
